Мій перший репозиторій.

Відповіді до завдань:
1. Git – розподілена система керування версіями написана мовою С. В основному це набір консольних утиліт, які відстежують і фіксують зміни в файлах.
2. Git має три основних стани, в яких можуть перебувати ваші файли: 1)Збережений у коміті (commited), змінений (modified) та індексований (staged).
3. Команда git add додає необхідний файл до репоиторію.
4. git status показує інформацію про поточний стан сховища: чи є актуальною інформація на ньому, чи немає чогось нового, що змінилося, і так далі.
5. git commit виконує фіксацію стану сховища в певний момент часу, тобто фіксація змін.
6. git init виконує процес ініціалізації Git у папці проекту.
7. В коментар комітів пишуть коментарі повними реченнями англійською мовою.
8. Щоб переглянути список коммітів необхідно вписати команду git log .
9. Команда git checkout дозволяє переміщатися між гілками, створеними командою git branch.
